Stilt Fox™ File Sorter is a software to help you manage the storage of reference materials. It will help you keep track of whatever kind of file you like. Trade in your cluttered folders for a tightly organized and searchable experience. Stilt Fox™ File Sorter also removes duplicates for you. This is great for:
  • Family vacation photos
  • Archived files
  • Web comic collections
  • Much much more...
 Stilt Fox™ C++ Rest API is a server program designed for education instead of actual use. With a rich wealth of comments, a well written readme, and minimalist code, Stilt Fox™ C++ Rest API is here to teach you about web sockets and C++ programming. If you're interested in this type of thing you should check it out. If you're following the Tutorial the comments start on main.cpp. That's right, even this website itself is open source! Go ahead and take a peak at what's going on behind the scenes. There are no helpful comments or anything, after all we want to run our website lean, however it's a fun look at what it takes to make a simple website and host it on github.

Here at Stilt Fox™ we value simplicity over fancy frameworks.
